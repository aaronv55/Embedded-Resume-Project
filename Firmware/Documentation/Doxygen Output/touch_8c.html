◆ touch_update_position()

void touch_update_position ( void  )

Get both x and y position from the LCD.

Parameters
[in]NONE
Returns
NONE
Note
This is meant to be used in a multi-threaded system, meaning that it should be called (serviced) every iteration of the containing for{} loop. The position will only update if a touch occurs, triggering the ISR and setting the touch_detection flag. If no touch has occurred, position (0,0) will be returned.
