Detailed Description

This file contains both low-level LCD drivers and a light set of graphics functions used to display bitmaps, text and other elements on the screen. It is specifically meant for the QD-Tech QD3502 LCD, which includes a built-in ILI9486 LCD controller.

Returns
current_byte Count so the main program can keep track of how many bytes have been sent since the file began
Note
