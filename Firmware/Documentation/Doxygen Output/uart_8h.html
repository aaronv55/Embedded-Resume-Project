◆ uart1_arduino_plotter()

void uart1_arduino_plotter ( char  temp_single_char)

Sends a string over USART directly to Arduino Serial Plotter.

Parameters
[in]temp_single_charSingle char to be send
Returns
NONE
Note
This function is only intended to interact with the "Serial Plotter" within the Arduino IDE. This plotter has an abnormal API, in which ASCII characters are converted to integers then displayed. For example, if the integer 123 was to be displayed, USART1 would need to send char '1', then char '2', then char '3'
