SUMMARY:
This directory contains various types of documentation for 
the .h/.c files which make up the project’s microcontroller firmware.

DOXYGEN:
Doxygen is a popular program used to create interactive, searchable
code documentation of an entire code base. In order to view the main
GUI for this project, open the file “index.html” inside of the 
subdirectory “Doxygen Output”.
